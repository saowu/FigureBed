#### 功能支持
```
1.支持多文件上传（set max=5）
2.支持导出csv上传路径
3.支持返回多种链接格式 （link, markdown, html, bbcode，removal）
4.可以通过removal链接自行删除图片文件
5.使用mysql数据库
6.支持拖动上传
```
